Elite Fitness Gym Website

 Overview

Elite Fitness Gym is a responsive website developed using HTML5, CSS3 and JavaScript. The purpose of the website is to provide information about gym memberships, fitness classes and contact details while maintaining accessibility and responsiveness across different devices.

---

 Design

 Aim

The aim of this website is to promote Elite Fitness Gym and provide users with information regarding memberships, classes and contact options.

 Objectives

 Provide information about gym services.
 Display available fitness classes.
 Allow users to contact the gym.
 Promote membership benefits.
 Provide an accessible and responsive user experience.

 User Stories

 As a visitor, I want to view information about the gym so that I can decide whether to join.
 As a visitor, I want to see available fitness classes so that I can find suitable training sessions.
 As a visitor, I want to contact the gym so that I can ask questions or request information.
 As a mobile user, I want the website to function correctly on smaller screens so that I can access information easily.

 Justification of Changes

Several changes were made following the original wireframe design.

 The navigation menu was simplified to improve usability and make navigation clearer.
 A contact form was added to allow users to send enquiries directly.
 A Google Maps section was added to help users locate the gym.
 Responsive design improvements were implemented to improve usability on mobile devices.
 Accessibility features such as alternative image text, labels and appropriate colour contrast were added to support users with accessibility requirements.

 Development

 Technologies Used

 CSS3

 Website Features

 Home Page

The homepage introduces the gym, provides information about memberships and encourages users to join.

 Classes Page

The classes page displays available classes, a timetable and frequently asked questions.

 Contact Page

The contact page contains a contact form, opening hours, location map and social media links.

 Responsive Design

CSS media queries were implemented to ensure the website adapts correctly to smaller screen sizes.

 Accessibility

Accessibility features include:

 Alternative text for images.
 Labels for form inputs.
 Semantic HTML structure.
 Appropriate colour contrast.
 Responsive design for different devices.

 Challenges Encountered

 Challenge 1

Creating a responsive navigation system.

 Solution

CSS media queries were implemented to adjust the layout on smaller devices.

 Challenge 2

Embedding external content such as Google Maps.

 Solution

Google Maps embed functionality was integrated using an iframe.

 Challenge 3

Maintaining accessibility while creating a visually appealing design.

 Solution

Accessibility guidelines were considered throughout development, including alt text, labels and colour contrast.
